# 实验方案：双目标对抗优化框架

## 1. 实验目标

生成对抗样本同时满足两个目标：
1. **说话人识别攻击**：使说话人识别系统将音频误分类为其他说话人
2. **净化鲁棒性**：使扩散模型净化失效，保持对抗扰动在净化后仍然有效

## 2. 数据集

### 主数据集
- **VoxCeleb1/2**: 说话人识别标准数据集
  - 训练集：用于训练说话人识别模型
  - 测试集：用于生成对抗样本（选择100-200个音频样本）
  
### 备选数据集
- **LibriSpeech**: 如果需要更多样化的语音数据
- **TIMIT**: 用于小规模快速验证

## 3. 基线模型

### 3.1 说话人识别模型
- **主模型**: ResNet-based speaker embedding (如 ResNet34)
- **备选**: ECAPA-TDNN, x-vector
- **预训练权重**: 使用VoxCeleb预训练模型

### 3.2 扩散净化模型
- **主模型**: Diffusion-based audio purification (基于 arXiv:2303.01507)
- **配置**: 
  - 扩散步数: T=50-100
  - 噪声调度: linear/cosine
  - 采样方法: DDPM/DDIM
- **备选**: 如果实现困难，可先用简单的去噪自编码器作为代理

## 4. 攻击方法实现

### 4.1 核心算法：双目标PGD

```
输入: 原始音频 x, 目标说话人 y_target, 扰动预算 ε
输出: 对抗样本 x_adv

初始化: x_adv = x
for i = 1 to N_iter:
    # 计算双目标损失
    L_speaker = CrossEntropy(SpeakerModel(x_adv), y_target)
    L_purification = ||SpeakerEmbed(x_adv) - SpeakerEmbed(Purify(x_adv))||
    L_total = α * L_speaker + β * L_purification
    
    # 梯度更新
    grad = ∇_{x_adv} L_total
    x_adv = x_adv + step_size * sign(grad)
    x_adv = clip(x_adv, x - ε, x + ε)
    
return x_adv
```

### 4.2 关键技术点

**净化感知梯度计算**：
- **方法1（快速近似）**: 使用单步或少步扩散采样近似净化过程
- **方法2（精确但慢）**: 使用完整扩散采样，通过重参数化技巧使其可微
- **推荐**: 先用方法1快速验证，再用方法2精调

**权重调整策略**：
- **固定权重**: α=0.5, β=0.5（baseline）
- **自适应权重**: 根据两个损失的相对大小动态调整
- **阶段性权重**: 前期侧重L_speaker，后期侧重L_purification

## 5. 超参数配置

### 5.1 攻击超参数
- **扰动预算 ε**: [0.01, 0.02, 0.05] (相对于音频幅度)
- **迭代次数 N_iter**: [50, 100, 200]
- **步长 step_size**: ε / N_iter * 2
- **损失权重 α, β**: 
  - 固定: **(0.6, 0.4)**（当前配置，优先声纹攻击）, (0.5, 0.5), (0.7, 0.3), (0.3, 0.7)
  - 自适应: 根据损失比例动态调整

### 5.2 净化模型超参数
- **扩散步数**: [10, 50, 100]
- **噪声强度**: [0.1, 0.3, 0.5]

## 6. 评估指标

### 6.1 主要指标（成功标准）
1. **说话人识别攻击成功率 (ASR)**: 
   - 定义: 对抗样本被误分类为目标说话人的比例
   - 目标: > 90%

2. **净化后对抗扰动保持率 (PPR - Post-Purification Robustness)**:
   - 定义: 净化后的对抗样本仍能成功攻击的比例
   - 目标: > 70%

### 6.2 辅助指标
3. **音频质量指标**:
   - SNR (Signal-to-Noise Ratio): 衡量扰动大小
   - PESQ/STOI: 衡量感知质量

4. **对抗鲁棒性**:
   - 对不同净化强度的鲁棒性
   - 对不同说话人识别模型的迁移性

### 6.3 对比基线
- **单目标攻击（仅攻击说话人识别）**: 不考虑净化
- **单目标攻击（仅破坏净化）**: 不考虑说话人识别
- **两阶段攻击**: 先攻击说话人识别，再增强净化鲁棒性

## 7. 实验流程

### Phase 1: 环境搭建与模型准备
1. 安装依赖: PyTorch, torchaudio, diffusers, speechbrain
2. 下载数据集: VoxCeleb1测试集（约100个样本）
3. 加载预训练模型:
   - 说话人识别模型
   - 扩散净化模型（或训练简单代理模型）

### Phase 2: 基线实验
1. 验证说话人识别模型准确率（应 > 95%）
2. 验证净化模型效果（对干净样本的影响 < 5%）
3. 运行单目标攻击基线

### Phase 3: 双目标攻击实验
1. **实验3.1**: 固定权重 α=β=0.5
   - 不同扰动预算 ε
   - 不同迭代次数
   
2. **实验3.2**: 权重消融实验
   - α=0.7, β=0.3
   - α=0.3, β=0.7
   - 自适应权重

3. **实验3.3**: 净化强度消融
   - 不同扩散步数
   - 不同噪声强度

### Phase 4: 对比与分析
1. 与单目标攻击对比
2. 与两阶段攻击对比
3. 可视化分析（频谱图、损失曲线）

## 9. 预期挑战与解决方案

### 挑战1: 扩散净化的可微分性
- **问题**: 完整扩散采样计算成本高且难以求导
- **解决**: 
  - 使用DDIM快速采样（10-20步）
  - 使用重参数化技巧使噪声采样可微
  - 或使用代理模型（去噪自编码器）快速验证

### 挑战2: 双目标优化的权重平衡
- **问题**: 两个损失的量级可能差异很大
- **解决**:
  - 损失归一化
  - 自适应权重调整
  - 多目标优化算法（如梯度归一化）

### 挑战3: 计算资源
- **问题**: 扩散模型推理慢，大规模实验耗时
- **解决**:
  - 先在小数据集（50-100样本）验证
  - 使用GPU加速
  - 并行化实验

## 11. 成功标准检查点

- ✅ **Checkpoint 1**: 基线模型验证通过（准确率 > 95%）
- ✅ **Checkpoint 2**: 单目标攻击成功（ASR > 90%）
- ✅ **Checkpoint 3**: 双目标攻击达到目标（ASR > 90% AND PPR > 70%）
- ✅ **Checkpoint 4**: 对比实验完成，证明双目标优于单目标

## 12. 备选方案

如果主方案遇到困难：
1. **简化净化模型**: 用简单的高斯去噪代替扩散模型
2. **降低成功标准**: ASR > 80%, PPR > 60%
3. **两阶段实现**: 先实现单目标攻击，再逐步加入第二目标
