# Project Truth

## 研究主题
语音对抗攻击：TTS防克隆 + 净化失效

## 研究目标
在普通的TTS防克隆（让说话人识别失效）之外，加上使得语音净化（purification）失效的能力

## 当前阶段
ideation（创新点生成）→ 已完成，准备进入实验阶段

## 核心创新方向
**双目标对抗优化框架**：设计统一的优化框架，生成对抗样本同时满足两个目标：
1. 使说话人识别系统将音频误分类为其他说话人
2. 使扩散模型净化失效，保持对抗扰动在净化后仍然有效

### 关键技术点
- 双目标联合优化：平衡说话人识别攻击损失和净化鲁棒性损失
- 净化感知梯度：在优化过程中模拟扩散净化过程
- 自适应权重调整：根据两个目标的达成情况动态调整损失权重

### 技术路径
使用PGD攻击框架，损失函数为 L = α·L_speaker + β·L_purification
- L_speaker: 使说话人识别失效
- L_purification: 通过可微分的扩散采样近似确保净化后仍保持对抗性
